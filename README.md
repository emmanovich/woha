# Test Data Generator

Test Data Generator creates random records that can be used while developing or testing applications.

The application generates names, addresses, phone numbers and dates, then exports the dataset to JSON and CSV formats.

---

## Features

- Random names
- Random addresses
- Random phone numbers
- Random dates
- JSON export
- CSV export

---

## Example

```
Generating records...

Records created: 5

John Miller
742 Oak Street
+1-555-2084
2024-05-16

Export completed.

JSON : output.json
CSV  : output.csv
```

Run

```bash
go run .
```

The project uses built-in sample dictionaries.
